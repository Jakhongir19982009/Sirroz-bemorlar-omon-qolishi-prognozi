# Sirroz-bemorlar-omon-qolishi-prognozi
Jigar sirrozi bilan og'rigan bemorlar ma'lumotlarini tahlil qilib, omon qolishi ehtimollikligiga yuqori ta'sir qiluvchi omillarni aniqlab, ehtimoliy tashxisni belgilab beruvchi Machine Learning model.

Ushbu vazifani bajarishda yuqori aniqlikka ega natijalarni olish uchun log-loss yo'qotishlarni kamaytirishga qaratilgan, kuchaytirilgan tasniflovchi XGBClassifier model algoritmidan foydalanildi.
